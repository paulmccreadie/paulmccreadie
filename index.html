I Know A Little Bit About Alot

Web development has long been a hobby of mine so I know a little bit about quite alot. I decided this year I'd seek to make my hobby my career by cranking up my skill level up to a commercial standard. Albeit entry level.

I've worked in sales in the commercial print sector, so I'd like to think my strongest skills are in design UI/UX and client liason. With some knowledge of marketing and branding I'm happy to discuss briefs with clients. In additon to my basic knowledge of HTML, SCSS and Javascript, I also have some knowledge of Spring boot - enough to configure MVC end points, connect to a database and host with SSL. A small amount of dev op knowledge, enough to create sub-domains, or email accounts. And tiny bit of PHP enough to install and edit a WP_theme.

I've a rudementary understanding of project planning, workflows and time management. And I'm a basic user of Sketch, Postman, Eclipse and Visual Studio Code. So when I decided, in January, to create this portfolio I started with a project plan.
